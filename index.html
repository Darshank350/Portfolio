Guess-My-Number

The guess-my-number game is a simple game where the player needs to guess a number between 1 to 20. If the player guesses the correct number, the score is added to the high score. The game is implemented using HTML, CSS, and JavaScript. The game provides a fun and engaging way for players to test their guessing skills.

Ping-Game

Pig game is a two-player game that involves rolling a die to score points. If a player rolls a 1, their turn ends and the other player gets a chance to play. Players can also choose to hold their score to add it to their total points. The first player to reach 100 points wins. The game is implemented using HTML, CSS, and JavaScript and provides a fun way to challenge your friends.
